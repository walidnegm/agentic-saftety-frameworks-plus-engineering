# Agentic Safety Plus: Engineering

*Agent-Native Engineering Practices for High-Velocity Software Factories Built on Layered Agent Safety.*

A living specification of the engineering-framework layer for software production where autonomous LLM-driven agents are first-class participants in the build process — drafting plans, adversarially reviewing each other's work, generating code in parallel pods, verifying artifacts at scale, surfacing drift, and orchestrating deploys. The human is the functional architect and thread coordinator, not the implementer.

The framework rests on four disciplines (PPRE, assume-compromise verification, behavior pinning, symptom-to-root) activated through eighteen specialized agents grouped into six lifecycle buckets, coordinated by a Central Factory Supervisor that enforces global invariants and routes work across concurrent streams (CDID — Concurrent Development, Integration, Deployment).
